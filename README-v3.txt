เวอร์ชัน v3 แก้หน้าแอดมินตรวจการชำระเงิน:
- เพิ่มปุ่มกลับในหน้ารายการ และหน้ารายละเอียด
- มือถือใช้การ์ดและมีปุ่ม "เปิดดูการชำระเงิน" เต็มความกว้าง
- คอมพิวเตอร์ยังใช้ตาราง
- แปลสถานะการชำระเงินเป็นภาษาไทยทั้งหน้ารวม หน้ารายละเอียด และฟอร์มตรวจสลิป
- ปรับปุ่มอนุมัติ/ปฏิเสธบนมือถือให้เรียงแนวตั้ง

วิธีรัน:
1. นำ .env.local จากโฟลเดอร์เดิมมาใส่
2. npm install
3. rm -rf .next
4. npm run dev
