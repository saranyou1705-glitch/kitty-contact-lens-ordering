แก้การเชื่อมสินค้าฝั่งแอดมินกับฝั่งลูกค้า

- หน้า Products ลูกค้าดึงข้อมูลล่าสุดผ่าน API และรีเฟรชทุก 5 วินาที
- การ์ดสินค้าแสดงจำนวน SKU ที่ Active และจำนวน SKU ที่มีสต๊อค
- รายละเอียดสินค้าแสดงทุก SKU Active แยกตามสีและค่าสายตา
- แสดง SKU และยอดสต๊อคพร้อมขาย
- SKU สต๊อค 0 แสดงว่า "หมด" และเลือกไม่ได้
- ตรวจจำนวนก่อนเพิ่มลงตะกร้า
- แก้ Excel Import ให้เปิด Color เดิมกลับเป็น Active
- การนำเข้า SKU Inactive หนึ่งบรรทัดจะไม่ปิดทั้ง Model โดยผิดพลาด
- หลัง Import ระบบคำนวณสถานะ Model จาก SKU Active ทั้งหมดใหม่

หมายเหตุ: SKU หลายตัวที่ใช้ Model Name เดียวกันจะแสดงเป็นการ์ด Model เดียวในหน้ารวม
เมื่อลูกค้ากดเข้า Model จะเห็นสี ค่าสายตา และ SKU ทั้งหมด
