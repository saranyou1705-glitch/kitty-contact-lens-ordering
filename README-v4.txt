ปรับหน้า Admin > ตรวจการชำระเงิน

- เอาการ์ดด้านบนออกทั้งหมด เหลือรายการเดียวแบบตาราง Responsive
- สถานะที่ไม่ทราบค่า เปลี่ยนเป็น "รอหลักฐานการโอน"
- หากมีสลิปแต่สถานะว่าง จะแสดง "รอตรวจสอบ"
- ปุ่มตรวจสอบอยู่ในหน้าจอเสมอ ไม่ต้องเลื่อนแนวนอน
- เลขออเดอร์แยกขึ้นคนละบรรทัด
- เปลี่ยนหัวข้อวันที่เป็น "วันที่อัปเดต"
- ใช้ payment_batches.updated_at และ fallback เป็น created_at
- บนโทรศัพท์ซ่อนคอลัมน์ที่ไม่จำเป็น แต่ยังเห็นยอดและเวลาอัปเดตในแถว
