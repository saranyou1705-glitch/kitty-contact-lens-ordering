แก้ 2 จุด:

1. ลูกค้าชำระแล้ว แต่หน้าแรกยังขึ้นว่ายังไม่ชำระ
- Dashboard และหน้ารายละเอียดออเดอร์ตรวจ payment_batches + payment_batch_orders เพิ่ม
- ถ้า payment batch เป็น approved/paid ระบบจะแสดง ชำระแล้ว
- ถ้ารอสลิปตรวจ ระบบจะแสดง รอตรวจสอบการชำระ
- คำนวณยอดชำระและยอดคงค้างจากข้อมูลล่าสุด

2. ปุ่มย้อนกลับซ้อนกัน
- เอาปุ่มย้อนกลับในหน้ารายละเอียดออเดอร์ออก
- เหลือปุ่มย้อนกลับกลางจาก CustomerNavigation เพียงปุ่มเดียว

วิธีรัน:
1. นำ .env.local จากเวอร์ชันเดิมมาใส่
2. npm install
3. rm -rf .next
4. npm run dev
