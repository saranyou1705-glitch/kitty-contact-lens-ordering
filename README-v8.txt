แก้ตามคำขอล่าสุด:

ลูกค้า Dashboard
- เปลี่ยนสถานะเป็น 5 กลุ่ม:
  1) กำลังดำเนินการ
  2) รอชำระเงิน
  3) รอตรวจยอด
  4) จัดส่งแล้ว
  5) สำเร็จแล้ว
- ทุกการ์ดกดเพื่อกรองออเดอร์ในสถานะนั้นได้
- จัดส่งแล้วไม่สนใจสถานะชำระเงิน
- สำเร็จแล้ว = ได้รับสินค้าแล้วและจ่ายเงินแล้ว หรือถูกปฏิเสธ/ยกเลิกโดยแอดมิน

ยืนยันรับสินค้า
- ในหน้าออเดอร์ที่จัดส่งแล้ว จะมีปุ่ม/ฟอร์มยืนยันรับสินค้า
- ลูกค้าติ๊กสินค้า กรอกจำนวนที่ได้รับ และแนบรูปสินค้า
- ถ้ารับสินค้าแล้วและชำระแล้ว ระบบจะนับเป็นสำเร็จแล้ว

ตะกร้าสินค้า
- ลดความสูงการ์ดสินค้า
- ลดความกว้างช่องจำนวน
- เหมาะกับกรณีมีสินค้าจำนวนมาก

หน้าสินค้า
- จำนวนสั่งซื้อกรอกตัวเลขได้โดยตรง พร้อมปุ่ม + / -

ปุ่มกลับ
- ย้ายปุ่มกลับไปลอยด้านล่างซ้ายเหนือเมนู ไม่ทับหัวข้อหน้า
