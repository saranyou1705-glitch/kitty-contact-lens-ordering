ปรับ UX ฝั่งลูกค้า:
- เอาเมนู "ออเดอร์" ออกจากแถบด้านล่าง
- /orders จะพากลับ /home
- หน้าแรกแสดงออเดอร์ทั้งหมด ไม่ตัดเหลือ 5 รายการ
- การ์ดตัวเลขบน Dashboard กดกรองออเดอร์ตามสถานะได้
- กดการ์ดเดิมซ้ำหรือกด "ดูทั้งหมด" เพื่อยกเลิกตัวกรอง
- หน้ารายละเอียดออเดอร์ /orders/[id] ยังใช้งานตามปกติ

วิธีรัน:
1. นำ .env.local จากโฟลเดอร์เดิมมาใส่
2. npm install
3. rm -rf .next
4. npm run dev
